## This is the project folder for PycnoPie, an experiment ran July-Aug 2024 in Sitka, AK on the indirect effects of Pycnopodia on urchin grazing for urchins from kelp forests and urchin barrens.

#### **Main analysis:** Run the following scripts/docs for main analysis for this paper, which is on the effect of Pycnopodia on the amount of kelp grazed by urchins from different source habitats

*\_\_main.R (00_kelp_cleaning -\> 01_kelp_manipulation):* loads all
neccessary libraries and cleans/prepares kelp data for analysis

*docs/PycnoPie_report.qmd* : runs main analysis of kelp grazing data,
including code for all figures included in the main text of the paper

**Supplemental:**

*scripts/animal_sizes.R:* Calculates average sizes of urchins/sea stars
used in experiments

Run numbered behavior scripts for behavior figures

**Other:**

*docs/kelp_analysis.qmd* is the initial report made to look at raw data
during the experiment

Project website can be found
[here](https://nikita-sridhar.github.io/PycnoPie/)
